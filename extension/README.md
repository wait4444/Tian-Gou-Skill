# 🐕 舔狗Skill — VS Code 扩展

极致忠诚的 AI 行为技能扩展。安装后自动将舔狗skill配置到你的工作区。

## 功能

- **一键安装**：打开任意项目，自动安装舔狗skill
- **状态栏指示**：实时显示 skill 激活状态
- **人格浓度可调**：全力/标准/轻量/专业 四档切换
- **功能开关**：独立控制情绪感知、惊喜彩蛋、护主预警
- **用户画像管理**：一键编辑/重置用户画像

## 命令

| 命令 | 说明 |
|------|------|
| `舔狗Skill: 安装到当前工作区` | 将 skill 安装到当前项目 |
| `舔狗Skill: 从当前工作区卸载` | 卸载 skill |
| `舔狗Skill: 查看状态` | 查看安装状态和配置 |
| `舔狗Skill: 编辑用户画像` | 打开用户画像文件编辑 |
| `舔狗Skill: 重置用户画像` | 重置画像为初始模板 |

## 设置

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| `tianGou.autoInstall` | `true` | 自动安装到新工作区 |
| `tianGou.personality` | `normal` | 人格浓度 |
| `tianGou.features.moodDetection` | `true` | 情绪感知 |
| `tianGou.features.surpriseExtras` | `true` | 惊喜彩蛋 |
| `tianGou.features.proactiveGuardian` | `true` | 护主预警 |
| `tianGou.features.selfDiscipline` | `true` | 自我鞭策 |
